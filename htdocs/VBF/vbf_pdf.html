PDF set choice

Several parton distribution functions (PDF) are available from different groups.
The order of the PDF evolution is specified in parenthesis. When a single order is specified, the LO-NLO-NNLO cross-section are computed with the same pdf. When more than one order is specified, then each cross-section at a given order is computed with the corresponding order PDFs. If no LO PDF is available, the LO cross-section is computed with the NLO PDF.
