VBF @ NNLO : Cross-Section Calculator
by P. Bolzoni, F. Maltoni, S.-O. Moch and M. Zaro
Higgs production in vector-boson fusion (VBF) is computed via a structure-function approach, as reported in arXiv:1003.4451 [hep-ph] and arXiv:1109.3717 [hep-ph].
In addition to the SM Higgs boson, it is also possible to compute the VBF cross-section for a generic neutral, single or double charged scalar or to a new vector that couples to the SM vector bosons.
This simple interface allows any registered user to obtain a cross section up to NNLO in QCD, including an estimate of the theoretical uncertaintes coming from scale variation and PDF uncertaintes.
The electro-weak parameters used for the cross-section computation are set to their respective PDG values (see the list here).
The code runs over the CP3-MadGraph cluster and might take up to a few hours depending on the actual request. An e-mail with a link to the results page is sent to the user as soon as results are available.
Please note that results for the production of a SM Higgs boson at the LHC, obtained with this code possibly also including EW corrections (as computed in arXiv:0707.0381 [hep-ph] and arXiv:0710.4749 [hep-ph]), can be found in the CERN Yellow Reports Handbook of LHC Higgs cross-section 1 and 3, as well as on the Higgs Cross-Section Working Group page.
