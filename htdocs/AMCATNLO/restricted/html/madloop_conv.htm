Conventions used by MadLoop:

  • μR = μF = QES =91.188 GeV

  • The finite part, single and double pole correspond to the coefficients ci in the following expression for the virtual amplitude V:
    V = (4π)ε/Γ(1-ε) (c-2/ε2 + c-1/ε + c0)

  • Regularization scheme: CDR

  • UV renormalization included (MSbar for massless modes, zero-momentum subtraction for massive ones)

  • Identical particle factor included

  • No complex mass scheme! Widths are set to 0


For more detail please look at the original MadLoop paper (arXiv:1103.0621)
