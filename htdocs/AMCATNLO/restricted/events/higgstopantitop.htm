Scalar and pseudoscalar Higgs production in association with a top-antitop pair

  • Date: 13 May 2011

  • Description:
      Event files for Higgs production in association with a top-antitop pair. Three scenarios are considered:

        1. scalar H, with Higgs mass 120 GeV;

        2. pseudoscalar A, with Higgs mass 120 GeV;

        3. pseudoscalar A, with Higgs mass 40 GeV.

      The Yukawa coupling is assumed to be SM-like, yt/sqrt(2) = mt/v. As renormalization and factorization scales the geometric mean of the transverse masses of the three final state particles has been used: μR = μF = (mTtmTtbarmTH/A)1/3, where mT = sqrt(m2+pT2) and the top pole mass has been set equal to the MSbar mass mt = 172.5 GeV. The bottom quark is treated massless (and its Yukawa coupling is set to zero) and the PDF set MSTW2008(N)LO has been used for the (N)LO.
  • Event files:
      All the NLO events are generated with the MC subtraction terms for Herwig 6 and the files contain 1 million events. The cross section is given between the brackets. As they are, the NLO events are still non-physical: they need to be showered with herwig 6 to be meaningful.

      • LHC, 7 TeV:
        scenario I. NLO (103.4 fb), LO (104.5 fb)
        scenario II. NLO (31.9 fb), LO (27.6 fb)
        scenario III. NLO (77.3 fb), LO (69.6 fb)

      • LHC, 14 TeV:
        scenario I. NLO (708 fb), LO (642 fb)
        scenario II. NLO (289 fb), LO (244 fb)
        scenario III. NLO (599 fb), LO (516 fb)

  • Technical details:
      The NLO event files use the UseSudakov=.true. and DampMCsubt=.true., using the default values of the code in revision 208. The event files were generated between 25-30 March 2011.
