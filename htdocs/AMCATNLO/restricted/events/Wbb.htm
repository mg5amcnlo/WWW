W-boson production in association with a bottom-antibottom pair

  • Description:
      Event files for W-boson production in association with a bottom-antibottom pair. The event files contain both W+ and W-, and include the decay of these W-bosons to W → e νe, keeping spin correlations and off-shell effects. The bottom quarks are kept massive everywhere and no cuts are applied to them. Input parameters are the following:

      • mW = 80.419 GeV,
      • ΓW = 2.0476 GeV,
      • αEW = 1/132.507 GeV,
      • mZ = 91.188 GeV,
      • mb = 4.5 GeV,
      • diagonal CKM matrix,
      • MSTW2008(N)LO_nf4 for the (N)LO,
      • μR = μF = mTW + (mTb + mTbbar)/2, where mT = sqrt(m2+pT2).
  • Event files:
      The NLO events are generated with the MC subtraction terms for Herwig 6 and the files contain 1 million events. The cross sections are given between the brackets, and the uncertainty is from the MC integration only. As they are, the NLO events are still non-physical: they need to be showered with herwig 6 to be meaningful.

      • Tevatron (no cuts): NLO (8.04 +/- 0.04 pb), LO (4.633 +/- 0.002 pb)

      • LHC 7 TeV (no cuts): NLO (38.68 +/- 0.08 pb), LO (19.37 +/- 0.01 pb)

      • LHC 7 TeV (pT,W > 100 GeV): NLO (1.080 +/- 0.003 pb)

  • Technical details:
      The NLO event files use the UseSudakov=.true. and DampMCsubt=.true., using the default values of the code in revision 208. A technical cut has been applied to the electron-neutriono invariant mass: events for which the invariant mass is more than 30 times the W-boson width away from its mass have been removed. All the events have the (off-shell) W-boson explicitly in the event record as intermediate particle, so its invariant mass should be conserved by Herwig. The event files without cuts were generated on March 23, 2011 and with the cut on the transverse momentum of the W-boson on April 2, 2011.
