Diphoton production in the ADD model

  • Description:
      Event files for the prompt production of di-photons in the ADD model. Both signal and backgrounds are included (pp → (Grav →) γγ). Input parameters are the following (unless specified below):

      • αEW-1 = 132.507,
      • GF = 1.16639 10-5 GeV-2,
      • mZ = 91.188 GeV,
      • MSTW(n)lo2008cl68 for the (N)LO,
      • μR = μF = mγγ (i.e. the diphoton invariant mass).
  • Generation cuts:
      • Two central photons, |η(γ)| < 2.6,
      • Two hard photons, pT(γ) > 20 GeV,
      • Large diphoton invariant mass, mγγ > 100 GeV or mγγ > 400 GeV (see the event files below),
      • Isolated photons, Frixione isolation with δ0 = 0.38, εγ = 1 and n = 2,
      • Remove the unphysical region: require mγγ < 1.1 MS, where MS is the ADD string scale.
  • Event files:
      The (N)LO events are generated with the MC subtraction terms for Herwig 6 (angular-ordered shower) and the files contain 1 million events each and are for LHC 8 TeV. The cross sections are given between the brackets, and the uncertainty is from the MC integration only. Furthermore, they contain information about the 40 MSTW PDF sets and 3x3 renormalization and factorization scales (0.5 and 2.0 times the central values) to compute the PDF and scale uncertainties. As they are, the NLO events are still non-physical: they need to be showered to be meaningful.

      • Dimension d = 2, MS = 3.7:

  • Technical details:
      The NLO event files use the UseSudakov=.true. and DampMCsubt=.true., using the default values of the FKS5_new_born code in revision 268. Analytic results (which have also been used for arXiv:0811.1670 and arXiv:0902.4894) have been used for the virtual corrections.
