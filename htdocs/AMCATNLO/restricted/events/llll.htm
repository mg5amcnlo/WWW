Production of four leptons

  • Description:
      Event files for the hadro-production of 4 leptons. All contributions to pp → llll have been taken into account. The dominant contributions are through on-shell Z- or W-bosons, but also photon exchange and single-resonant contributions are included (with all interferences and spin-correlations). Input parameters are the following:

      • mZ = 91.188 GeV,
      • ΓZ = 2.4414 GeV,
      • ΓW = 2.0476 GeV,
      • αEW = 1/132.507 GeV,
      • mW = 80.419 GeV,
      • MSTW(n)lo2008cl68 for the (N)LO,
      • μR = μF = mllll (i.e. the 4-lepton invariant mass)
      • Cut on all opposit-charged same-flavour dilepton invariant masses: ml+l- > 30 GeV.
  • Event files:
      The (N)LO events are generated with the MC subtraction terms for Herwig 6 (angular-ordered shower) or Pythia 6 (Q2-ordered shower) and the files contain 1 million events each and are for LHC 7 TeV. The cross sections are given between the brackets, and the uncertainty is from the MC integration only. Furthermore, they contain information about the 40 MSTW PDF sets and 3x3 renormalization and factorization scales (0.5 and 2.0 times the central values) to compute the PDF and scale uncertainties. As they are, the NLO events are still non-physical: they need to be showered to be meaningful.

  • Technical details:
      The NLO event files use the UseSudakov=.true. and DampMCsubt=.true., using the default values of the code in revision 291. Only events what were produced via an integration channel for which the diagram has a Z- or W-boson relatively close to its on-shell mass, this intermediate resonance is written in the event record. Therefore, for some events the invariant mass of the lepton pair is not kept fixed by Herwig.
