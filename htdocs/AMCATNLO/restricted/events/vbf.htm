Higgs boson production in vector boson fusion (VBF)

  • Description:
      Event files for Higgs boson production in VBF, at the 8 TeV LHC. The Higgs boson is not decayed in the event files. Input parameters are the following:

      • mH = 125 GeV,
      • αEW = 1/132.507 GeV,
      • GF = 1.16639 10-5 GeV-2,
      • mZ = 91.188 GeV,
      • ΓZ = 2.496 GeV,
      • ΓW = 2.089 GeV,
      • diagonal CKM matrix,
      • MSTW2008NLO pdfs,
      • μR = μF = mW (computed from the other parameters)
      • For numerical stability reasons, events are generated requiring at least two anti-kt jets with pT > 2 GeV, R=0.5
  • Event files:
      The NLO events are generated with the MC subtraction terms for Herwig 6, Pythia 6 (virtuality ordered) and Herwig++. Each event file contains 1 million events. The NLO cross-section is 1.633 +/- 0.003 pb (the uncertainty is from the MC integration only). The Herwig 6 sample contains weight information to obtain scale and PDF uncertainties on the fly. As they are, the NLO events are still non-physical: they need to be showered with the corresponding MC in order to give meaningful results.

      • Herwig 6

      • Pythia 6

      • Herwig++

  • Technical details:
      The NLO event files use the UseSudakov=.true. and DampMCsubt=.true.. All interferences (e.g. between identical final state quarks) are included in the computation, with the only exception of the virtuals (pentagon diagrams have not been included, as they are known to be completely negligible).
