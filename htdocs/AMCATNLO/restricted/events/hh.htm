Higgs boson pair production in gluon fusion

  • Description:
      Event files for Higgs boson pair production gluon fusion. The Higgs boson is not decayed in the event files. Input parameters are the following:

      • mH = 125 GeV,
      • αEW = 1/132.507 GeV,
      • GF = 1.16639 10-5 GeV-2,
      • mt = 173 GeV,
      • mb = 0 GeV (5 flavour scheme is used, Hbb couplings are neglected),
      • diagonal CKM matrix,
      • MSTW2008NLO (errors at 68\% CL) pdfs,
      • μR = μF = mHH/2 (halt the invariant mass of the higgs boson pair)
  • Event files:
      All samples contains weight informations to obtain scale and PDF uncertainties on the fly. As they are, the NLO events are still non-physical: they need to be showered with the corresponding MC in order to give meaningful results.

      • LHC 8 TeV NLO Herwig6 (σ=8.73 fb)

  • Technical details:
      NLO event files use the UseSudakov=.true. and DampMCsubt=.true.. Events are weighted, as a reweighting procedure has been applied to include the top-quark loop in the n- and (n+1)-body kinematics.
