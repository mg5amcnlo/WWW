Z-boson production in association with a bottom-antibottom pair

  • Description:
      Event files for Z/γ*-boson production in association with a bottom-antibottom pair. They include the decay of the Z/γ*-bosons to Z/γ* → e+e- keeping spin correlations and off-shell effects. The bottom quarks are kept massive everywhere and no cuts are applied to them. Input parameters are the following:

      • mZ = 91.188 GeV,
      • ΓZ = 2.4414 GeV,
      • αEW = 1/132.507 GeV,
      • mW = 80.419 GeV,
      • mb = 4.5 GeV,
      • mt = 172.5 GeV,
      • MSTW2008(N)LO_nf4 for the (N)LO,
      • Cut on the dilepton invariant mass me+e- > 30 GeV
      • μR = μF = mTe+e- + (mTb + mTbbar)/2, where mT = sqrt(m2+pT2).
  • Event files:
      The NLO events are generated with the MC subtraction terms for Herwig 6 and the files contain 1 million events. The cross sections are given between the brackets, and the uncertainty is from the MC integration only. As they are, the NLO events are still non-physical: they need to be showered with herwig 6 to be meaningful.

      • Tevatron: NLO (1.509 +/- 0.006 pb), LO (0.860 +/- 0.001 pb)

      • LHC (7 TeV): NLO (15.94 +/- 0.05 pb), LO (9.64 +/- 0.01 pb)

  • Technical details:
      The NLO event files use the UseSudakov=.true. and DampMCsubt=.true., using the default values of the code in revision 228. Only events what were produced via an integration channel for which the diagram has the Z-boson this intermediate resonance is written in the event record. Therefore, for some events the invariant mass of the lepton pair is not kept fixed by Herwig. Furthermore, the so-called Hagiwara contributions (a finite and gauge independent set of diagrams formed by closed fermion loops attached to the EW-vector boson) have not been included. For the V-events Monte-Carlo over helicities has been used. The event files were generated on May 11, 2011.
