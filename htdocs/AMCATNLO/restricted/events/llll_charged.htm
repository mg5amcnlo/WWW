Production of 4 charged leptons

  • Description:
      Event files for the hadro-production of 4 charged leptons. All contributions to pp → l+l-l+l- have been taken into account. The dominant contributions are through two on-shell Z-boson (pp → ZZ → l+l-l+l-), but also photon exchange and single-resonant contributions are included (with all interferences and spin-correlations). For the gg-induced channels the Higgs boson was excluded from the process. Input parameters are the following:

      • mZ = 91.188 GeV,
      • ΓZ = 2.4414 GeV,
      • αEW = 1/132.507 GeV,
      • mW = 80.419 GeV,
      • MSTW(n)lo2008cl68 for the (N)LO (also the NLO set for the gluon-induced process),
      • μR = μF = ml+l-l+l- (i.e. the 4-lepton invariant mass)
      • Cut on all opposit-charged same-flavour dilepton invariant masses: ml+l- > 30 GeV.
  • Event files:
      The (N)LO events are generated with the MC subtraction terms for Herwig 6 (angular-ordered shower) or Pythia 6 (Q2-ordered shower) and the files contain 1 million events each and are for LHC 7 TeV. The cross sections are given between the brackets, and the uncertainty is from the MC integration only. Furthermore, they contain information about the 40 MSTW PDF sets and 3x3 renormalization and factorization scales (0.5 and 2.0 times the central values) to compute the PDF and scale uncertainties. As they are, the NLO events are still non-physical: they need to be showered to be meaningful.

      • e+e-μ+μ-:
        • NLO Herwig 6 (15.562 +/- 0.014 fb)
        • NLO Pythia 6 (Q2-ordered) (15.543 +/- 0.013 fb)
        • LO (11.134 +/- 0.007 fb)

      • e+e-e+e-:
        • NLO Herwig 6 (6.413 +/- 0.007 fb)
        • NLO Pythia 6 (Q2-ordered) (6.424 +/- 0.007 fb)
        • LO (4.577 +/- 0.005 fb)

      • gg → e+e-μ+μ- (this is an event file with weighted events):
        • Loop-induced gg channel (0.691 +/- 0.002 fb)

      • lllν, llνν, lννν and νννν:
        • Click here for a seperate page on the four leptons of which at least one is a neutrino.

      The (N)LO event files below are after showering. They only contain information on the 4 leptons, so only observables that depend on leptons can be plotted. The default MC tunes were used, without underlying event, multiple interaction or QED showers.

      • e+e-μ+μ-:
        • NLO Herwig 6
        • NLO Pythia 6 (Q2-ordered)
        • LO Herwig 6
        • LO Pythia 6 (Q2-ordered)

      • e+e-e+e-:
        • NLO Herwig 6
        • NLO Pythia 6 (Q2-ordered)
        • LO Herwig 6
        • LO Pythia 6 (Q2-ordered)

      • gg → e+e-μ+μ- (these are event files with weighted events):
        • Loop-induced gg channel Herwig 6
        • Loop-induced gg channel Pythia 6 (Q2-ordered)

  • Technical details:
      The NLO event files use the UseSudakov=.true. and DampMCsubt=.true., using the default values of the code in revision 291. Only events what were produced via an integration channel for which the diagram has a Z-boson relatively close to its on-shell mass, this intermediate resonance is written in the event record. Therefore, for some events the invariant mass of the lepton pair is not kept fixed by Herwig.

      The loop-induced gg-channels have been computed by generating an unweighted event sample with 1M Born events (qq → e+e-μ+μ-, but with gluon PDFs and without single-resonant contributions) using MadGraph v5. These events have been reweighted against the loop-induced process by MadLoop. To remove numerical instabilities, the event file was splitted in 500 seperate files which were combined into 1 final file assinging a relative weight to each event file inversely proportional to the fluctuations of the weights in that particular event file (computed using the standard deviation).
