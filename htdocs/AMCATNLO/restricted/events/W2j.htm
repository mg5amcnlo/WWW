W+ + 2 jet production

  • Description:
      Event files for the production of a W+-boson in association with two hard jets. The decays of the W-boson to positrons and neutrinos are included (with spin correlations and off-shell effects). Input parameters are the following:

      • mW = 80.419 GeV,
      • ΓW = 2.0476 GeV,
      • αEW = 1/132.507 GeV,
      • mZ = 91.188 GeV,
      • diagonal CKM matrix,
      • MSTW2008(N)LO for the (N)LO,
      • μR = μF = HT/2 (Scalar sum of transverse momentum of all final state particles at the matrix element level (including neutrino) divided by two),
      • jets are defined with anti-kT algorithm with R = 0.4 (and minimum pT defined below)
  • Event files:
      The NLO events are generated with the MC subtraction terms for Herwig 6 and the files contain 10 million events and are for Tevatron. They contain information about the 40 MSTW PDF sets and 3x3 renormalization and factorization scales (0.5 and 2.0 times the central values) to compute the PDF and scale uncertainties. As they are, the NLO events are still non-physical: they need to be showered with herwig 6 to be meaningful.

      • pT(j) > 5 GeV: NLO Herwig

      • pT(j) > 10 GeV: NLO Herwig (has only 9999999 events)

      When plotting distributions, both event files should be used. To have unbiased results, the distributions generate with the two event files should coincide (within statistical fluctuations). If not, jet cuts at the analysis level should be made harder. See also the W1j page
  • Technical details:
      The NLO event files use the UseSudakov=.true. and DampMCsubt=.true., using the default values of the code in revision 251. The event files were generated in July, 2011.
