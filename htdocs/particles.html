Standard Model (full information, calculator)

The Standard Model of electroweak and strong interactions. Different versions are available: with or whithout Higgs boson, with identity CKM matrix or with non zero Cabibbo angle.
Note that the parameters for the new MG5 models differ from those in the old MG4 version.

Model files:

particles.dat interactions.dat (w/o higgs, with ckm) all model files on CVS (w/o higgs, with ckm)

Minimal Supersymmetric Standard Model (full information, calculator)

Standard MSSM with no R parity violation.
Note that the MG5 model is using the SLHAII parameter card format, while the MG4 model is using SLHAI.
